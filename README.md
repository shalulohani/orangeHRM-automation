\# 🧪 OrangeHRM Automation with Playwright



!\[Playwright CI](https://github.com/shalulohani/orangeHRM-automation/actions/workflows/playwright-ci.yml/badge.svg)



End‑to‑end automation framework for \*\*OrangeHRM\*\* using \*\*Playwright + TypeScript\*\*.  

This project demonstrates modern QA practices, reusable page objects, and CI/CD integration via \*\*GitHub Actions\*\*.



\---



\## 🚀 Features

\- Playwright + TypeScript setup  

\- Page Object Model (POM) structure  

\- Allure \& Playwright HTML reports  

\- CI pipeline with GitHub Actions  

\- Scalable test suite for HRM modules  



\---



\## 🧩 Project Structure



orangeHRM-automation/

│

├── pages/                # Page objects

├── tests/                # Test specs

├── playwright.config.ts  # Playwright configuration

├── .github/workflows/    # CI pipeline

└── README.md             # Project overview



\---



\## ⚙️ CI/CD Workflow

The pipeline runs automatically on every push or pull request to `main`.  

It installs dependencies, executes tests, and uploads reports as artifacts.



\---



\## 📊 Reports

\- \*\*Playwright HTML Report:\*\* `playwright-report/`

\- \*\*Allure Results:\*\* `allure-results/`
